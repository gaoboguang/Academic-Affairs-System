## 7. 当前数据风险

`backend:data-health` 仍然显示 P0 数据可用性缺口。这些不是代码启动失败，但会影响高考数据和推荐结论的完整性。

当前 6 条主要缺口：

1. 特殊类型已有招生计划，但缺少专门录取结果：春季高考、艺术类、体育类、单独招生、综合评价招生。
2. 山东 2024 年招生计划数量偏少，仍需核验完整性。
3. 一分一段缺少 2020-2023 年。
4. 省控线 / 批次线缺少 2020-2023 年。
5. 政策参考数量偏少，当前仅 4 条。
6. 招生章程限制链仍有 1748 条待人工复核。

因此，普通类推荐可作为当前主链路试用；特殊类型只能作为初筛和人工核对顺序参考，不能包装成录取把握。

## 8. Mac 上怎么启动

普通用户优先使用：

```bash
cd /Users/gao/local-edu-tool
npm run start:local
```

或在项目根目录双击：

```text
start-local-edu.command
```

默认打开地址：

- 前端：`http://127.0.0.1:5173`
- 后端健康检查：`http://127.0.0.1:8000/api/system/health`
- API 文档：`http://127.0.0.1:8000/docs`

停止后台服务：

```bash
cd /Users/gao/local-edu-tool
npm run stop:local
```

开发调试时再使用：

```bash
npm run dev
```
