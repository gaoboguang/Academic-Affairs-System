# 山东 2026 高考数据发布状态与导入观察清单

- 日期：2026-04-25
- 执行窗口：B2，2026 已公开招生数据监控与导入入口
- 范围：山东生源地；普通类夏季高考优先；单招 / 综评只做已公开政策与计划边界登记

## 1. 状态口径

本轮不新增重复业务表，先把 2026 状态下沉到 `backend:data-health` 和 `/gaokao-data`：

| 状态 | 含义 |
| --- | --- |
| `published` | 官方已公开来源，但还没有完成结构化导入 |
| `imported` | 当前主库已有可查询记录 |
| `pending_official_release` | 还在等待官方正式发布 |
| `not_applicable` | 当前阶段本来不会产生，例如 2026 投档结果 |
| `manual_review_required` | 来源分散或附件需人工下载、核验、登记 |

## 2. 2026 当前可用数据

| 数据项 | 当前状态 | 处理方式 |
| --- | --- | --- |
| 2025/2026 适用选科要求 | 已有来源登记；主库已有 2026 选科字典时显示 `imported` | 推荐时必须作为专业候选硬约束 |
| 单招 / 综评政策通知 | `published`；来源为山东省教育厅 2025-12-22 通知 | 只能解释单招 / 综评政策边界 |
| 单招 / 综评院校计划限额 | `published` 或 `manual_review_required` | 官方附件需人工下载、登记、核验后再解析 |
| 高校单招 / 综评章程与分专业计划 | `manual_review_required` | 高校官网分散发布，先走人工导入，不做不稳定批量抓取 |

已登记官方来源：

- 山东省教育厅《关于做好2026年高职（专科）单独考试招生和综合评价招生工作的通知》：`https://edu.shandong.gov.cn/art/2025/12/22/art_107093_10344338.html`
- 山东省教育招生考试院 2025/2026 适用选科要求：`https://www.sdzk.cn/NewsInfo.aspx?NewsID=6819`

## 3. 2026 当前不能当成完整数据的数据

| 数据项 | 当前状态 | 红线 |
| --- | --- | --- |
| 普通类夏季高考正式招生计划 | `pending_official_release`，除非主库已经有 2026 普通类计划记录 | 不得用单招 / 综评计划限额替代普通类正式计划 |
| 2026 普通类投档 / 录取结果 | `not_applicable` | 投档录取后才会产生，填报前不得伪造最低分、最低位次 |
| 2026 一分一段表 | `pending_official_release` | 未发布前可用 2025 一分一段临时估算，但必须提示“按上一年估算” |
| 2026 各类别分数线 / 省控线 | `pending_official_release` | 未发布前只能参考历史资格线，不能替代院校或专业录取结果 |

## 4. 人工导入入口

先登记默认来源和目录：

```bash
npm run backend:gaokao-sources -- --json
```

查看 2026 来源文档编号：

```bash
npm run backend:gaokao-sources -- --list --json
```

如果官方附件自动抓取不稳定，把文件放到：

```text
data/imports/gaokao/official/2026/
data/imports/gaokao/manual/2026/
```

然后登记文件：

```bash
npm run backend:gaokao-import-official -- \
  --source-document-id <2026来源ID> \
  --file data/imports/gaokao/manual/2026/<官方附件文件名> \
  --importer-name shandong_single_comprehensive_plan_limit \
  --json
```

当前 B2 只保证登记、批次、健康状态和页面提示。单招 / 综评计划限额的结构化解析应在后续专项窗口补齐；在解析完成前，页面必须保持“需人工核验”语义。

## 5. 推荐页提示口径

2026 普通类推荐可以先使用：

- 2023-2025 普通类历史投档数据；
- 2023-2025 一分一段和分数线；
- 2025/2026 适用选科要求；
- 已公开的 2026 政策或计划边界，但只能作为边界提示。

推荐结果必须提示：

```text
2026 普通类正式招生计划如未完全公开，当前推荐主要基于 2023-2025 历史投档数据和已公开规则，正式填报前必须以山东省教育招生考试院最终发布的招生计划和高校章程为准。
```
