## 七、项目功能总结

这个项目当前可以概括为：

> 一个面向高中场景、运行在本地 `127.0.0.1`、单用户、离线优先、中文后台界面的教务决策工具。

## 7.1 已具备的主要能力

- 本地维护学校基础台账
- 管理学生、教师和任教关系
- 创建考试、导入成绩、生成成绩分析
- 导入课表、修正映射、计算工作量
- 管理成长档案与附件
- 管理院校、专业、录取数据
- 生成普通生与艺体生推荐结果
- 导入评教数据并生成汇总
- 记录班主任量化明细与附件
- 统一导出多类 Excel 报表
- 执行本地备份、恢复与审计追踪

## 7.2 当前技术形态

- 前端：Vue 3 + TypeScript + Vite + Element Plus
- 后端：FastAPI + SQLAlchemy + SQLite + Alembic
- 本地数据目录：`data/`
- 前端页面数：`14`
- 后端测试文件数：`12`
- 当前后端测试：`21` 通过

如果把状态分成四档：

1. 仅有骨架
2. 核心功能初步可用
3. 主要业务闭环完成，可稳定试用
4. 全量增强项和工程化全部收尾

我对这个项目当前的判断是：

> **处于第 3 档，接近第 4 档，但还没有完全到第 4 档。**

## 8.2 为什么这样判断

因为它已经满足：

- P0 全部核心业务

- 本地运行、导入、分析、推荐、导出、备份恢复闭环

但还没有满足：

- P2 增强项只完成了小部分
- 更完整的核心业务 E2E 覆盖
- 桌面打包

如果继续推进，建议优先顺序是：

1. 继续扩充 Playwright 对更复杂对比、多方案和失败回退的回归
