# 山东升学方案中心使用说明

- 日期：2026-04-25
- 入口：侧栏“升学方案”或学生详情页“升学方案”
- 适用范围：山东生源地学生的升学路径初筛、材料核对和普通类推荐入口

## 1. 这个页面能做什么

山东升学方案中心用于帮助老师先判断学生可以重点关注哪些升学路径：

- 普通类常规批。
- 普通类提前批 A / B。
- 普通类特殊类型批。
- 春季高考本科 / 专科。
- 高职单独招生。
- 高职综合评价招生。
- 艺术类本科 / 专科。
- 体育类常规批。
- 体育单招。
- 高水平运动队。

页面会把每条路径整理成卡片，说明：

- 当前是否适合继续关注。
- 只能资格初筛，还是可以进入完整位次推荐。
- 缺哪些画像字段或材料。
- 哪些要求必须人工核验。
- 下一步应先做什么。

## 2. 推荐操作顺序

1. 打开“升学方案”。
2. 选择学生和目标年份，默认目标年份是 2026。
3. 如果画像摘要里有“未维护”，先点“编辑画像”。
4. 在学生详情页补齐“升学画像”，再回到“升学方案”刷新。
5. 先看“材料缺口”，补影响路径最多的材料。
6. 再看“升学路径卡片”，重点关注“适合关注 / 可能适合 / 信息不足 / 需人工复核”。
7. 普通类常规批可点“进入普通类推荐”继续查看冲 / 稳 / 保候选。
8. 需要给学生或家长沟通时，点“打印报告”或“导出 Excel”。

## 3. 学生升学画像怎么填

画像字段越完整，路径初筛越清晰。常用字段包括：

- 生源地：当前主要按山东判断。
- 考生类型：普通类、春考、艺术、体育、中职、社会人员等。
- 考试类型：夏季高考、春季高考、艺术类、体育类、单招、综评等。
- 选科组合：普通类、提前批、特殊类型路径会用到。
- 春考专业类别、艺术类别、体育专项。
- 是否完成高考报名。
- 是否普通高中应届、中职学生、社会人员。
- 是否有高中阶段毕业证书或同等学力。
- 是否接受专科、民办、中外合作、省外、提前批、定向服务、面试体检政审。
- 身体限制、家庭约束、地域偏好、职业偏好和备注。

缺项不代表学生不符合条件，只表示系统无法自动判断，需要补充事实或人工复核。

## 4. 怎么读路径状态

| 状态 | 含义 |
| --- | --- |
| 适合关注 | 当前画像和材料支持继续看这条路径 |
| 可能适合 | 有一定匹配，但仍有材料或风险要补 |
| 信息不足 | 画像或材料不够，暂时不能判断 |
| 需人工复核 | 需要核对官方公告、报名系统或高校章程 |
| 不建议 | 当前画像已明显不匹配该路径 |
| 当前不适用 | 该路径不适合当前学生或当前年份 |

## 5. 推荐深度是什么意思

| 推荐深度 | 含义 |
| --- | --- |
| 可接完整位次推荐 | 当前只用于普通类常规批，可进入山东普通类冲稳保推荐 |
| 资格初筛 | 只判断报名资格、身份、类别、材料和基础门槛 |
| 政策提醒 | 主要提醒报名时间、测试安排、章程和人工核验事项 |
| 必须人工复核 | 系统只能列清单，不能自动判断 |

除普通类常规批外，其他路径不要理解为录取概率。

## 6. 报告怎么用

“打印报告”会打开打印预览页，可在浏览器里打印或保存为 PDF。

“导出 Excel”会生成一份工作簿，包含：

- 汇总页。
- 学生画像。
- 路径建议。
- 材料缺口。
- 下一步行动。
- 数据风险。

这份报告适合用于学生谈话、家长沟通和校内讨论。正式填报前仍要逐条核验山东省教育招生考试院公告、山东省教育厅通知、目标高校招生章程和当年正式计划。

## 7. 当前不能做什么

- 不能伪造 2026 普通类正式招生计划。
- 不能伪造 2026 一分一段、省控线或投档结果。
- 不能把校内考试名次直接当成山东全省位次。
- 不能把单招、综评、春考、艺体、体育、提前批、特殊类型、体育单招或高水平运动队说成录取概率。
- 不能用单招 / 综评计划限额替代普通类夏季高考正式计划。
- 不能替代高校招生章程、体检、政审、面试、单科、语种、身高、视力、专项资格等人工核验。

## 8. 常见情况

### 路径显示“信息不足”

先看该卡片里的“缺失材料”，再到学生详情页“升学画像”补字段。补完后回到升学方案中心刷新。

### 普通类常规批可以推荐，为什么还有数据风险

当前主要参考 2023-2025 历史投档数据。2026 普通类正式计划、一分一段、省控线仍需要等官方发布后再导入，所以系统必须继续提示风险。

### 单招、综评、春考为什么不给冲稳保

这些路径依赖身份、报名系统、专业类别、测试方式、院校章程和分专业计划。当前系统只做资格初筛和材料清单，不做完整录取把握。

### 体育类、体育单招、高水平运动队有什么区别

体育类常规批、体育单招、高水平运动队的报名系统、资格条件、考试方式和录取规则不同，页面会分开显示，不能混用。

### 报告能不能直接给最终填报结论

不能。报告是路径规划和材料核对工具，正式填报结论必须以当年官方发布和目标高校章程为准。
