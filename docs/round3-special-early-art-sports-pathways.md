# 第三轮 D7 艺体、体育、提前批、专项路径初筛

- 日期：2026-04-25
- 范围：窗口 D7，艺术类、体育类、普通类提前批 A/B、普通类特殊类型批、体育单招、高水平运动队路径初筛
- 后端入口：`apps/backend/app/services/gaokao_pathways.py`
- 前端入口：学生详情页“升学画像”、`/gaokao-pathways`
- 本地装载命令：`npm run backend:bootstrap-pathways -- --target-year 2026`

## 目标

D7 在 D1-D6 的路径模型、官方来源、学生画像、方案中心、普通类推荐加固和单招/综评/春考初筛基础上，继续细化艺体、体育、提前批和专项类路径。

本轮继续保持边界：这些路径只输出资格初筛、规则说明、材料缺口和人工复核清单，不输出录取概率；体育类常规批、体育单招、高水平运动队必须分开展示，不混用报名系统、测试方式和录取规则。

  - 普通类、艺术类同批次兼报限制。
  - 与体育单招、高水平运动队区分的风险提示。

## 官方来源口径

D7 继续复用 D2 已登记的官方来源：

- 山东省教育招生考试院艺术类招生工作实施方案。
- 山东省教育招生考试院体育类专业招生有关工作通知。
- 山东省教育招生考试院普通高校招生录取工作意见。
- 山东省教育招生考试院民航飞行技术招生通知。
- 教育部特殊类型招生工作通知。
- 国家体育总局体育单招管理办法。

这些来源只用于路径规则、材料提醒和人工复核清单，不代表系统已结构化导入目标高校全部章程、测试细则、资格名单或录取结果。

## 页面读法

- `艺术类本科/专科`：先看艺术类别与成绩材料，再核对文化线、综合分、章程限制和兼报边界。
- `体育类常规批`：先看体育身份、高考报名和体育测试成绩，再核对综合分、批次线、章程和兼报限制。
- `普通类提前批 A/B`：先看提前批意向、体检面试政审接受度和定向服务接受度，再逐校核对章程、协议和专项要求。
- `普通类特殊类型批`：先看控制线、资格材料、报名审核、公示和校测安排；不能把特殊类型初筛当作普通类常规批推荐。
- `体育单招`：重点核对等级证书、专项报名系统、文化/专项考试和院校简章。
- `高水平运动队`：重点核对等级材料、高校资格审查、测试、公示、文化成绩和招生项目。

这些路径出现 `信息不足` 或 `需人工复核` 是正常状态，表示还要补事实或查章程，不表示学生一定不能走该路径。
