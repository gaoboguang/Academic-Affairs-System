## 5. 招生计划处理

本轮发现并登记 2023-2025 招生计划补充信息页面和 docx 附件直链，但本机访问山东省教育招生考试院 `Floadup` 附件时持续 SSL / 超时失败。

这些文件仍需人工下载后放入 `data/imports/gaokao/official/{year}/`，再运行：

```bash
./.venv/bin/python scripts/round5_register_plan_supplement_documents.py --no-download --json
```

补充信息不能替代完整《填报志愿指南》，也不能关闭 2024 招生计划数量偏少缺口。
